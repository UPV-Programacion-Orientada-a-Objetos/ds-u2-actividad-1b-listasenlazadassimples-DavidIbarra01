Detailed Description

template<typename T>
class ListaSensor< T >

Implementa una Lista Enlazada Simple Genérica.

Esta clase gestiona la memoria (nodos) de forma manual y cumple con la Regla de los Tres para un manejo seguro de punteros y memoria dinámica.

Template Parameters
TEl tipo de dato que almacenará la lista (ej. int, float, SensorBase*).
