\doxysubsection{Detailed Description}
Abstrae la comunicación con el puerto serial (ej. Arduino). 

Utiliza las APIs de bajo nivel de POSIX (\doxylink{structNodo_ae96eebd81fdcc626a95c8f4357a1c5b3}{termios.\+h}, \doxylink{structNodo_ae96eebd81fdcc626a95c8f4357a1c5b3}{fcntl.\+h}) para configurar y leer desde un puerto serial.
